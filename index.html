Web 101

Class 1

In this class you will learn the fundamentals of designing and building a web page. You will learn the primary technologies involved in building web pages (HTML, CSS, and Javascript). We will also introduce your project.

Topics Covered

  • Intro to Web
  • The DOM - The Basic HTML Tags
  • Rendering
  • Linking JavaScript and CSS - What is the difference between Inline and Linked CSS?
  • Separation of Concerns - Content vs Presentation vs Functionality

Class 2

The majority of the second class is dedicated to teaching you everything you need to know about CSS, focusing on commonly-used properties and selectors. By the end of class you will be able to follow along with the instructors in building a page layout.

Topics Covered

  • CSS Syntax and Selectors
  • The IMG Tag
  • Inline vs. Inline Block Display - Know the difference between inline and block elements and when to use each attribute for the CSS Display property
  • A Simple CSS layout - You should be able to build it yourself

Class 3

In this class you will gain some experience with various tools available to you as a web developer. There will also be a brief overview on JavaScript and what it does for web pages, but you won't have to know how to use it beyond a simple function or two.

Topics Covered

  • Browser Dev Tools - Be able to spend some time in the browser and not just a text editor
  • How to use Git with Eclipse - Pushing and Pulling
  • Intro to JS - document.getElementById

Class 4

In this class you will review everything you learned in the class and have the opportunity to ask the instructor questions. After that, students will present their website pages to the class.

Topics Covered
