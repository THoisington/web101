CODEcamp - Web 101

Course Overview

Why should I take Web Basics 101?

This entry level course teaches students essential skills in HTML (HyperText Markup Language) and CSS (Cascading Style Sheets). The techniques taught are those used by professional Web developers and provide students with a strong foundation to build their skills and create robust and scalable Web sites. Whether you want to develop a Web presence for your business or find a new career in Web design, this course will get you started.

What will I learn?

This course will teach you how to create multi-column Web page layouts that include components such as images, menus, lists and tables. Along the way, students will gain an understanding of how professional websites are developed and learn about the applications used to develop them.

What can I expect?

Each 2.5-hour session is a mixture of Lecture and hands-on Exercises to best teach you the fundamentals of Web design. Hands-on practice is essential to understand and fully grasp the concepts presented. The class is limited to 14 students.
